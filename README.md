PromptForge transforms vague ideas into production-ready prompts through a structured process of clarification, variant generation, and competitive testing (Arena).

## üöÄ Why PromptForge?

Most prompt tools focus on **quantity** (libraries of snippets). PromptForge focuses on **quality** via engineering:

1.  **Quality over Quantity:** We generate 3 distinct competitive variants for every problem.
2.  **Human-in-the-Loop:** You are the judge. Our "Arena" lets you test prompts side-by-side.
3.  **Model Agnostic:** Works with OpenAI, Anthropic, and Local LLMs (via Ollama).
4.  **Secure:** Your API keys are encrypted at rest.

## ‚ö° Quick Start (Docker)

The easiest way to run PromptForge is using Docker.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/promptforge.git
    cd promptforge
    ```

2.  **Configure Environment:**
    Copy the example configuration:
    ```bash
    cp .env.example .env
    ```
    *Optional:* Edit `.env` to set your ports or pre-load API keys.

3.  **Run with Docker Compose:**
    ```bash
    docker-compose up -d --build
    ```

4.  **Access the App:**
    *   **Frontend:** [http://localhost:3000](http://localhost:3000)
    *   **Backend Docs:** [http://localhost:8000/docs](http://localhost:8000/docs)
